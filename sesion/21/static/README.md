# Node APIRest

## Clone the repo

```
git clone https://github.com/jestrade/api-make-it-real-2021.git
```

## Install the app

```
yarn
```

## Create .env file

```
HTTP_PORT=
LOG_ACCESS=
JWTKEY=
APIWEATHERKEY=
DB_CONNECTION_STRING=
SALT_ROUNDS=
```

## Run the app

```
yarn start
```

## Dev mode

### Install nodemon globally

```
yarn install --global nodemon
```

### Run the app

```
yarn run dev
```

### Run tests

```
yarn run test
```
